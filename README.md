# ProgramacaoPraWeb
Repositório direcionado a matéria de Pratica de Programação web ministrada pelo professor Álvaro Lopes Rios

## Ideia
Foi desenvolvida uma página principal para que possa servir de caminho para vários projetos diferentes que forem sendo desenvolvidos ao decorrer da máteria sem ser necessário a criação de multiplos repositórios.

## Projetos em desenvolvimento
Na pasta curriculo encontra-se o primeiro projeto desenvolvido, com foco em ser um curriculo pessoal.
